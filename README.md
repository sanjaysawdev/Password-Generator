# Password Generator

A simple and responsive Password Generator built using React.

## Features
- Generate secure passwords
- Adjustable password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Copy password to clipboard

## Tech Stack
- Html
- React
- JavaScript
- CSS

## Project Structure
├── index.html
├── script.jsx
├── style.css
├── package.json
└── .gitignore

## Installation

Clone the repository:

git clone https://github.com/sanjaysawdev/Password-Generator.git

Move into the project folder:

cd Password-Generator

Install dependencies:

npm install

Run the project:

npm start
